SHELL: ANALYTICS PLATFORM

This is a re-design project for the client Shell, to create a visual hierarchy on their current analytics platform: TIBCO Spotfire X. The re-design includes their Global Performance Dashboard and the Process Safety Overview.

ROLE:

User Experience, User Interface, Research, Visual Design

This was done with my current internship as an UX Designer at SoftVision. I was able to take on the lead design role and re-create a style guide for Shell to follow. I also had to consider the flow of information architecture, as well as a styleguide for Shell.

TOOLS:

Sketch, Invision, Zeplin

CHALLENGE:

How can we create an experience that will allow the client to navigate through data with ease, with the consideration of the limitations of the TIBCO Spotfire X software.
