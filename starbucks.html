STARBUCKS: VOICE DESIGN AND AUGMENTED REALITY

This is an extended application for Starbucks with multiple features inlcuding AR, Voice, a Pull-to-thaw system, and a re-design of their Inventory platform.

ROLE:

User Experience, User Interface

This was done with my current internship as an UX Designer at SoftVision. I took on the lead design role and created interfaces for a Planogram using AR, a new Pull-to-thaw platform as well as a re-design of their current Inventory platform. I also mapped out a Voice interface using Dialogflow for Google Assistance.

TOOLS:

Sketch, Invision, Zeplin, Dialogflow
