Starbucks: Voice Design and Augmented Reality

*As part of the project NDA, I can't go into details about the overall details of the project. Feel free to reach out to me to learn more about the process!

Overview

This is an extended application for Starbucks with multiple features inlcuding AR, Voice, a Pull-to-thaw system, and a re-design of their Inventory platform.

Role:

User Experience, User Interface

Tools:

Sketch, Invision, Zeplin, Dialogflow

I took on the lead design role and created interfaces for a Planogram using AR, a new Pull-to-thaw platform as well as a re-design of their current Inventory platform. I also mapped out a Voice interface using Dialogflow for Google Assistance.
